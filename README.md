# Практические задания по использованию Windows API

Этот проект содержит 4 задания по работе с унаследованным кодом и Windows API.

## Требования

- Windows операционная система
- Компилятор C++ с поддержкой Windows API (MinGW, MSVC, или другой)
- Windows SDK

## Компиляция

### Использование Makefile (MinGW)

```bash
make
```

### Ручная компиляция (MinGW)

```bash
g++ -std=c++11 -o assignment1.exe assignment1.cpp -mwindows
g++ -std=c++11 -o assignment2.exe assignment2.cpp -mwindows
g++ -std=c++11 -o assignment3.exe assignment3.cpp -mwindows
g++ -std=c++11 -o assignment4.exe assignment4.cpp -mwindows
```

### Компиляция с MSVC (Visual Studio)

```bash
cl /EHsc assignment1.cpp /link /SUBSYSTEM:WINDOWS user32.lib
cl /EHsc assignment2.cpp /link /SUBSYSTEM:WINDOWS user32.lib
cl /EHsc assignment3.cpp /link /SUBSYSTEM:WINDOWS user32.lib
cl /EHsc assignment4.cpp /link /SUBSYSTEM:WINDOWS user32.lib
```

## Описание заданий

### Задание 1: Hello, World!

Простое приложение, которое отображает сообщение "Hello, World!" с помощью функции `MessageBox` из Windows API.

**Запуск:**
```bash
./assignment1.exe
```

### Задание 2: Игра "Угадай число"

Приложение, в котором пользователь загадывает число от 0 до 100, а компьютер угадывает его, используя бинарный поиск. Использует `MessageBox` для взаимодействия с пользователем. Предусмотрена возможность повторной игры.

**Запуск:**
```bash
./assignment2.exe
```

**Как играть:**
1. Загадайте число от 0 до 100
2. Отвечайте на вопросы компьютера:
   - **Да** - если компьютер угадал
   - **Нет** - если ваше число больше предложенного
   - **Отмена** - если ваше число меньше предложенного
3. После угадывания можно сыграть еще раз

### Задание 3: Закрытие окна Блокнота

Приложение ищет окно Блокнота с помощью функции `FindWindow` и отправляет ему сообщение о закрытии через `SendMessage`.

**Запуск:**
```bash
# Сначала откройте Блокнот
./assignment3.exe
```

**Примечание:** Перед запуском программы необходимо открыть Блокнот (Notepad).

### Задание 4: Обновление заголовка Блокнота

Приложение находит окно Блокнота и обновляет его заголовок текущим временем каждую секунду, используя функции `FindWindow` и `SetWindowText`.

**Запуск:**
```bash
# Сначала откройте Блокнот
./assignment4.exe
```

**Примечание:** 
- Перед запуском программы необходимо открыть Блокнот (Notepad)
- Программа будет работать до тех пор, пока окно Блокнота не будет закрыто
- Для остановки программы закройте окно Блокнота

## Используемые Windows API функции

- `MessageBox` - отображение диалоговых окон
- `FindWindow` - поиск окна по имени класса или заголовку
- `SendMessage` - отправка сообщений окну
- `SetWindowText` - изменение текста заголовка окна
- `IsWindow` - проверка существования окна
- `Sleep` - задержка выполнения программы

## Очистка

Для удаления скомпилированных файлов:

```bash
make clean
```

или вручную удалите все `.exe` файлы.
